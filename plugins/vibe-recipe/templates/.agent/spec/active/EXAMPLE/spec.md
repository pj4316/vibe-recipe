## 사용자 시나리오

### US-001 Example

- Actor: 사용자

- Independent test: acceptance를 독립적으로 검증할 수 있다.

Acceptance:

- AC-001: Given 준비된 조건, when 사용자가 행동한다, then 기대 결과가 발생한다.

# Triage Labels

engineering skills는 다섯 가지 canonical triage 역할을 사용합니다. 이 파일은 그 역할을 이 저장소의 실제 issue tracker 라벨 문자열에 매핑합니다.

| Label in mattpocock/skills | Label in our tracker | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | maintainer가 이슈를 평가해야 함 |
| `needs-info` | `needs-info` | reporter의 추가 정보가 필요함 |
| `ready-for-agent` | `ready-for-agent` | AFK agent가 바로 처리할 수 있을 만큼 명세됨 |
| `ready-for-human` | `ready-for-human` | 사람의 구현 또는 판단이 필요함 |
| `wontfix` | `wontfix` | 처리하지 않기로 결정함 |

스킬이 역할을 언급하면, 이 표의 오른쪽 라벨 문자열을 사용합니다.
